# Oasis_Infobyte_task2_BMI_Calculator
Python BMI Calculator with Tkinter GUI, BMI classification, input validation, and history tracking.
🧮 BMI Calculator with History (Python Desktop Application)
🚀 Excited to share my BMI Calculator with History project, developed using Python and Tkinter. This desktop application helps users calculate their Body Mass Index (BMI) based on height and weight, classifies their health status, and maintains a history of previous calculations.
✨ Key Features
✅ Calculate BMI accurately using standard formula
✅ Classify BMI (Underweight, Normal, Overweight, Obese)
✅ User-friendly Graphical User Interface (GUI)
✅ Input validation for accurate calculations
✅ Save BMI records automatically
✅ View previous calculation history
✅ Clean and interactive desktop application
🛠️ Technologies & Tools
🐍 Python
🖥️ Tkinter (GUI)
📂 File Handling (CSV/Text)
🧩 Object-Oriented Programming (OOP)
⚡ Event-Driven Programming
💻 Visual Studio Code
